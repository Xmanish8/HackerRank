# 🚀 Running the Code

Clone the repository:

```bash
git clone https://github.com/Xmanish8/HackerRank-Java-Solutions.git
```

Navigate to a problem:

```bash
cd Introduction/"Java Loops II"
```

Compile:

```bash
javac Solution.java
```

Run:

```bash
java Solution
```
